## 📝 Description
This is a secure PowerShell-based security script designed to prevent unauthorized access by deploying a Brute Force Protection mechanism. It monitors user login attempts and implements a strict account lockout policy after multiple failed authentication phases.

## ⚙️ Features
* *Secure Admin Credentials:* Hardcoded secure admin authentication mechanism.
* *Smart Policy Configuration:* Limits login attempts to a maximum of 3.
* *Live Attempt Tracker:* Displays real-time remaining attempts with visual warnings.
* *Account Lockout Phase:* Instantly triggers a critical security lockdown on 3 failed attempts.

## 🚀 How to Run
1. Open *PowerShell* on Windows.
2. Navigate to the folder containing brute_force_protection.ps1.
3. Run the script using:
   powershell
   .\brute_force_protection.ps1
